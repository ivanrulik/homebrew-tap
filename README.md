# homebrew-tap

Personal Homebrew tap for [@ivanrulik](https://github.com/ivanrulik)'s macOS apps.

## Install

```sh
brew install --cask ivanrulik/tap/<cask-name>
```

Or tap once, then install by short name:

```sh
brew tap ivanrulik/tap
brew install --cask <cask-name>
```

## Available casks

| Cask | Description |
|---|---|
| [`piano-book-pro`](Casks/piano-book-pro.rb) | Menu-bar app that turns your typing into piano. ([source](https://github.com/ivanrulik/piano_book_pro)) |
